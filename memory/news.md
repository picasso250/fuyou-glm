# 2026年2月26日 全球科技要闻

## 1. Nvidia Q4财报亮眼，AI支出热潮持续
Nvidia公布Q4营收同比增长73%，达到681.3亿美元，数据中心业务营收增长75%至623亿美元。Q1业绩指引超预期。分析显示大型科技公司AI支出已突破6300亿美元。

## 2. C3.ai宣布大规模裁员
企业AI公司C3.ai计划裁减约26%的员工（截至2025年4月员工数约1200人）。因全年业绩指引疲软，股价盘后下跌超20%。

## 3. Anthropic推出定时任务功能
Anthropic在Cowork中推出"定时任务"功能，允许Claude在特定时间自动完成重复性任务，标志着AI助手向自动化工作流程迈進。

## 4. Alphabet旗下Intrinsic并入Google
Alphabet的工业机器人AI公司Intrinsic正式并入Google，将作为独立实体继续运营并与Google DeepMind合作，推进机器人AI技术。

## 5. 三星发布Galaxy S26系列
三星Unpacked 2026发布Galaxy S26 Ultra（定价$1,299+）、S26及Buds 4 Pro，搭载OneUI 8.5系统。

## 6. Gemini支持Android多步骤自动化
Google为Android上的Gemini推出多步骤任务自动化功能，可自动完成叫车、外卖等复杂操作，并扩展了通话诈骗检测功能。
